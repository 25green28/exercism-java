---

---

This repository contains my solutions to the Java track on Exercism.

## Disclaimer

These solutions are shared for learning and reference purposes only.  
Feel free to use them, but please do **not** copy-paste them directly when completing exercises on Exercism.  
Try to understand the solutions and the reasoning behind them instead.

## Project Structure

The solutions are organized into Java packages following standard Java conventions.  
Each package name matches the original exercise name from Exercism, and the files inside
use the class names expected by the platform.
